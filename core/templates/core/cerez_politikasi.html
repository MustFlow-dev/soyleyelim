Çerez Politikası

1. Çerez (Cookie) Nedir?

Çerezler, ziyaret ettiğiniz web siteleri tarafından tarayıcınız aracılığıyla cihazınıza veya ağ sunucusuna depolanan küçük metin dosyalarıdır.

2. Kullanım Amacımız

Platformumuzun işlevselliğini artırmak, size özel içerikler sunmak ve site trafiğini analiz etmek amacıyla çerezler kullanmaktayız.

3. Çerez Türleri
  • Zorunlu Çerezler: Sitenin çalışması için gereklidir.
  • İşlevsel Çerezler: Tercihlerinizi hatırlamamızı sağlar.
  • Analitik Çerezler: Ziyaretçi davranışlarını analiz etmemize yarar.
4. Çerez Tercihlerinizi Nasıl Yönetebilirsiniz?

Tarayıcı ayarlarınızı değiştirerek çerezlere ilişkin tercihlerinizi kişiselleştirme imkanına sahipsiniz.
